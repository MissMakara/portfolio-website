This is a one page client portfolio website
It was written with HTML, CSS, Javascript, Bootstrap and python

The index file holds all the html and Js code 
The assets folder holds the css and other supporting js scripts
The flask_email_api file is the python script

The contact form writes an email to the client from interested users. 
This was powered using a python flask API and Javascript.

To launch it:
Install the requirements in the requirements.txt file
Run the flask_email_api.py file (python3 flask_email_api.py)

Then launch the index.html file
